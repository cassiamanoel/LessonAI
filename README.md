# 🚀 LessonAI: O Seu Criador de HQs Educativas (Marvel Grade)

LessonAI é um motor avançado de geração de Histórias em Quadrinhos (HQs) que utiliza Inteligência Artificial para transformar temas educativos em narrativas visuais cinematográficas. O projeto segue o padrão profissional **Marvel/DC Grade v9.0**, garantindo qualidade editorial, lettering autêntico e diagramação profissional.

## 🌌 Universo da História
O LessonAI foi concebido para ser um multiverso educacional. Os personagens (como robôs exploradores, cientistas interdimensionais ou heróis históricos) viajam através de conceitos complexos (como Ciclo da Água, Inteligência Artificial ou História do Brasil), explicando-os através da ação e do drama das HQs.

O tom é **Educativo, Épico e Cinematográfico**.

## 🛠️ Como Gerar Novas Páginas
1.  **Inicie o App**: No terminal, execute `uv run streamlit run app.py`.
2.  **Escolha o Tema**: Na barra lateral ou no campo de texto, defina o que deseja ensinar.
3.  **Ajuste o Script**: O Agente Roteirista gerará a quantidade de páginas solicitada (1-10). Você pode revisar e editar diálogos ou descrições visuais.
4.  **Gere a Arte**: Clique em "Gerar Imagens". O sistema criará a arte pura e depois aplicará o letreramento (balões de fala, pensamento, grito, etc.).
5.  **Exporte**: Ao final, vá para a tela de **Exportação** e gere o arquivo PDF consolidado.

## 📂 Estrutura de Documentação
- `prompt.md`: Contém as diretrizes mestres de geração e estilo.
- `personagens.md`: Detalhamento dos heróis, seus designs e personalidades.
- `roteiro/`: Pasta destinada ao armazenamento dos scripts (JSON/TXT) de cada capítulo.
- `paginas/`: Pasta para salvar as composições finais de imagens e PDFs.
